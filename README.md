# 🏏 Cricket Balls Store

A modern, responsive e-commerce front-end application designed for cricket enthusiasts. This project features a dynamic product catalog with real-time filtering and a functional shopping cart system, showcasing clean UI/UX design and efficient DOM manipulation.

## 🚀 Live Demo
**[View the Live Website][(https://muhammadhuzaifafarrukh.github.io/Cricket-Ball-Store/)](https://cricketballstore.vercel.app/)**

## ✨ Key Features
* **Dynamic Product Catalog:** Automatically renders a collection of premium cricket balls (Kookaburra, Duke, SG).
* **Multi-Criteria Filtering:** Users can instantly filter products by **Brand** (Kookaburra, Duke, SG) and **Color** (Red, White, Pink) without page reloads.
* **Real-time Search:** A search bar that filters the product grid as you type, matching names, brands, or colors.
* **Interactive Shopping Cart:** An in-memory cart system that updates the count in the navigation bar and provides user feedback when items are added.
* **Responsive UI:** Optimized for all devices, featuring a mobile-friendly burger menu and a grid layout that adapts to screen size.
* **Modern Aesthetics:** Includes a hero section with high-quality calls to action and a "Used by the Best" gallery featuring legendary cricketers.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom styling using CSS Variables for consistent theming, Flexbox, and Grid for layouts.
* **JavaScript (ES6+):** Pure Vanilla JS for all dynamic logic, including product filtering, search functionality, and cart management.
* **FontAwesome:** For intuitive iconography.
